1. Add your project image (JPG, PNG, or WebP)
2. Rename to match the filename (e.g., `islamic-history-series.jpg`)
3. Delete the `.svg` placeholder
4. Update file extension in `/app/page.tsx`:
   - Line 223: Featured project
   - Lines 240, 248, 256, 264: Smaller projects
