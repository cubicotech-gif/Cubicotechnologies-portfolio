# Hero Section Images

This folder contains preview card images for the homepage hero section.

## Current Files:

1. **dashboard-preview.svg** - LMS dashboard screenshot (800x600px)
2. **interactive-lessons.svg** - Interactive lesson screenshot (800x600px)
3. **analytics-preview.svg** - Analytics dashboard screenshot (800x600px)

## How to Replace:

1. Add your image (JPG, PNG, or WebP recommended)
2. Rename to match the filename (e.g., `dashboard-preview.jpg`)
3. Delete the `.svg` placeholder
4. Update file extension in `/app/page.tsx` line 183-185

## Recommendations:
- **Size:** 800x600px
- **Format:** JPG or WebP
- **Max file size:** 200KB each
- **Content:** Screenshots of your actual products/features
